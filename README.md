# Football Performance Analysis

Projet d’analyse de performance football à partir des open data StatsBomb.

## Objectif
Analyser la performance individuelle d’un joueur et la performance collective d’une équipe sur un match marquant de l’Euro 2024.

## Stack
- Python
- pandas
- numpy
- matplotlib
- mplsoccer
- statsbombpy

## Structure
- `data/` : données brutes et transformées
- `src/` : code source
- `notebooks/` : exploration et analyses
- `reports/` : figures et résultats
- `tests/` : tests unitaires
